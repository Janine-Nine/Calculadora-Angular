# Calculadora-Angular
Calculadora Cientifica feita com Angular

# 🧮 Calculadora Cientifica Angular App

Aplicação web de calculadora científica desenvolvida em Angular, com interface moderna e funções matemáticas avançadas, permitindo realizar operações básicas e científicas de forma rápida e intuitiva.

O projeto foi criado com foco em aprendizado de Angular, componentização, manipulação de eventos e organização de estrutura de aplicação front-end.

---

Operações matemáticas básicas (+, -, *, /)
Potência (xʸ)

Quadrado (x²)
Inverso (1/x)
Seno (sin)
Cosseno (cos)
Tangente (tan)
Logaritmo (log)
Logaritmo natural (ln)
Troca de sinal (±)
Número decimal

Backspace (⌫)
Interface responsiva
Layout moderno com CSS
